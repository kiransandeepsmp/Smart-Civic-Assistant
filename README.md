# Smart-Civic-Assistant
A responsive civic engagement web application that helps citizens explore government schemes, welfare programs, public resources, and interact with an AI-powered chatbot using the Gemini API.
Overview

Civic Engagement Portal is a responsive web application designed to help citizens access important government schemes, welfare programs, and civic resources easily. The project also includes an AI-powered chatbot integrated with the Gemini API to assist users interactively.

Features
AI-powered chatbot using Gemini API
Government schemes information portal
Civic awareness platform
Responsive and modern UI
Category-wise navigation
Interactive chatbot interface
Easy-to-use design
